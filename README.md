# The Nordic Buffet Calculator

An interactive web calculator for [The Nordic](https://thenordic.com/) ($145/person all-you-can-enjoy buffet in Charlestown, RI) that helps diners figure out how much they need to eat to get their money's worth.

## Features

- **46 menu items** across 6 categories (Sea, Raw Bar, Land, Soups, Sides, Desserts)
- **Restaurant vs. Supermarket pricing** toggle to compare what you'd pay elsewhere
- **Real price sources** from 5 nearby RI restaurants and 5 local markets, with per-item citations
- **Live scoreboard** with progress bar tracking your plate's value vs. the $145 buffet cost
- **Share results** — copy your plate breakdown to share with friends
- **Responsive design** — works on desktop and mobile
- **Dark mode** — respects system preference

## How It Works

Add items to your plate using the +/- buttons. The calculator tracks the total value of everything you've eaten based on real restaurant or supermarket prices, and tells you whether you've gotten your money's worth.

## Tech

Single self-contained HTML file. No build step, no dependencies, no framework.

- Google Fonts (Great Vibes, Playfair Display, Inter)
- CSS custom properties with light/dark mode
- Vanilla JavaScript
